# Змейка

Браузерная игра «Змейка», написанная на чистом HTML, CSS и JavaScript — без сборщиков, фреймворков и внешних зависимостей.

**Демо:** https://nurgysa.github.io/snake-game/

## Возможности

- Три уровня сложности (лёгкий, обычный, сложный) с разной скоростью и стартовым числом препятствий.
- Прогрессия по уровням: с ростом счёта уровень повышается, а вместе с ним добавляются новые препятствия.
- Локальные рекорды хранятся отдельно для каждой сложности.
- Препятствия на поле, при столкновении с которыми игра заканчивается.
- Золотая звезда — бонусная еда, дающая +3 очка без увеличения длины змейки.
- Синтезированные звуковые эффекты и лёгкие анимации для еды и бонуса; движение подчиняется настройке уменьшения анимации браузера или ОС, звук можно отключить.

## Управление

- Стрелки (`↑ ↓ ← →`) или `W A S D` — смена направления.
- Свайп по игровому полю — смена направления с телефона.
- `Enter` — начать игру или продолжить после паузы.
- `R` — начать новую игру.
- Кнопка «Звук» на экране — включить или выключить звуковые эффекты.

## Локальный запуск

Зависимостей нет — игра работает как статический сайт. Игра использует ES-модули, поэтому открывать `index.html` напрямую через `file://` может не сработать в некоторых браузерах. Запустите локальный сервер, например:

```
python -m http.server 4173
```

и откройте `http://localhost:4173`.

## Рекорды

Рекорды сохраняются в `localStorage` браузера отдельно для каждой сложности и не передаются на сервер.
